SQL project focused on retail sales analysis. Performed revenue analysis, customer spending analysis, product performance analysis, ranking, data cleaning, and monthly trend analysis using SQL.

# Retail Sales Analysis Using SQL

## Project Overview

This project analyzes retail sales data using SQL to identify revenue trends, top customers, top products, and business performance.

---

## Tables Used

- Customers
- Products
- Categories
- Orders

---

## SQL Skills Used

- SELECT
- JOIN
- GROUP BY
- ORDER BY
- CASE
- SUM
- AVG
- COUNT
- RANK()
- LAG()
- Data Cleaning Functions

---

## Analysis Performed

### Product Revenue Classification

Used CASE statements and aggregations to classify products as:

- High Revenue
- Medium Revenue
- Low Revenue

### Customer Spending Analysis

Calculated total spending by each customer.

### Customer Ranking

Used RANK() window function to identify top spending customers.

### Monthly Revenue Trend Analysis

Used LAG() to compare revenue with the previous month and identify:

- Growth
- Decline
- No Previous Data

---

## Key Insights

- Laptop generated the highest revenue.
- Ravi Darji was the highest spending customer.
- Revenue declined in February.
- Revenue recovered in March.
- Revenue declined again in April.
